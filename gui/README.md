# gui/

TypeScript + HTML5 Canvas graphic client for Zappy.

No game engines. Connection path and rendering land in Graphic-track tickets.

## Lint / test

```bash
npm install
npm run lint
npm test
```

Or from repo root: `./scripts/check.sh`

See [`AGENTS.md`](../AGENTS.md) and [`docs/SDS.md`](../docs/SDS.md).
